# Windows 系统安全：内核驱动开发

## 内容简介

本书系统地讲解Windows内核驱动程序的开发，内容偏向网络与系统安全防护。本书首先详细讲解驱动工程项目的创建，以及目前最新的KMDF内核驱动开发框架的使用。在基础内核驱动编程方面，有涉及如常用数据结构（内核链表等）、Unicode字符串处理、同步异步机制和应用层通信等。在系统安全防护方面，内容有涉及内核注册表和文件操作与监控、键鼠过滤驱动和内核CNG密码学编程。在网络安全方面，有涉及目前最新的NDIS协议驱动、LWF、WFP和WSK。

以Windows内核驱动开发为主题的书已经多年没有出版过了，且很多都只讲解了硬件设备的驱动开发，而忽略了内核驱动在系统与网络安全层面的应用。即使有少数类似书籍，但它们内容老旧，使用WDM框架，而本书完全摒弃这种已经官方放弃维护的框架，转而全篇使用KMDF。内核密码学编程矿建CNG、LWF和WSK等框架几乎没见过已出版的书中有讲解。本书还各个知识点有结合完整的可直接编译的代码，尽可能使用内核编程中允许的C++语法，而不是传统的C语言代码，测试环境和截图均使用极新的64位预览版Windows 11，摒弃已经过时的技术。

本书通过大量可直接编译运行的模板级代码讲解了Windows内核主机防护技术，适合Windows系统安全开发者、安全研究员和红队人员阅读，学习EDR与IDS开发技术。该书还大量讲解了现代化Windows内核驱动开发框架的使用，对硬件设备驱动的开发和游戏反作弊等系统的开发也有重要参考意义。

## 未完工内容

希望对本书内容感兴趣的读者能给本Github仓库一个Star，作者也会在考研后继续进行下面的各项完善，届时读者们可以在本仓库获得最新版的内容。

*   第一章环境配置章节的大量截图稍有麻烦，因为涉及VMware Workstation的安装、测试系统虚拟机的安装、Visual Studio的安装、WinDbg与Visual Studio双机调试等。
*   Minifilter章节暂时缺少Communication Port相关内容，应用层部分缺少卷附加的详细讲解。
*   NDIS相关章节缺少应用层的驱动安装的讲解，缺少NDIS微型端口驱动的讲解。 
*   打算讲解使用Rust语言编写KMDF驱动程序，但由于目前wdk\-build的默认时间戳服务器遭到关停，而作者没能找到合适的TOML文件的配置方法，这部分内容暂停推进。 
*   内存管理和PE文件解析等方面有些零碎内容还未添加，偏向内核攻防方面的内容在后期也会挑选可在生产环境可稳定使用的部分进行讲解，如HEVD、SSDT枚举等，各类Hook由于生产环境下防护加强，大抵是不打算涉及。 
*   Capstone在内核中的使用，以及ELAM和TrEE，作者在技术上遇到了困难，解决后可能会添加。 
*   添加Windbg的常用调试技巧、添加EWDK的使用，可能会添加企业级驱动的测试与发布流程。 
*   Sysinternals套件中DebugView在2026年3月左右进行了界面的更新，需要将书中前半部分输出截图进行替换。 
*   目录安排目前比较意识流，后期会慢慢调整。本电子书用$\LaTeX$编写，书籍排版源码暂不打算公开，但PDF中有诸多排版不当问题，如图片大小不合适、部分内容超出页面右侧等，需要进一步调整。
